## IPChanger
#### 一条命令切换系统代理

使用方法
```
usage: IPChanger.py [-h] [-e] [-d] [-p PROXY]

optional arguments:
  -h, --help            show this help message and exit
  -e, --enable          set proxy
  -d, --disable         disable proxy
  -p PROXY, --proxy PROXY
                        assign proxy
```

注意: 当代理为自动获取时，再次启用前，应先禁用，请勿重复启用
